# School-System-SQL
Student Management System using SQL with joins, aggregate functions and data analysis queries.

## Project Description
This project manages student, course, and marks data using SQL for data retrieval and performance analysis.

## Features
- Created STUDENTS, COURSES, and MARKS tables
- Used joins to combine multiple tables
- Performed queries for student performance analysis
- Applied aggregate functions

## SQL Concepts Used
- SELECT queries
- JOIN operations
- GROUP BY and aggregate functions
- Subqueries

## Learning Outcome
- Learned multi-table relationships
- Improved SQL query writing skills
- Data analysis using SQL
